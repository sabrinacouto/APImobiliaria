# 🏡 APImobiliária 🏡

## 💬 Rodando localmente

Clone o projeto

```bash
  git clone https://https://github.com/sabrinacouto/APImobiliaria.git
```

Entre no diretório do projeto

```bash
  cd APImobiliaria
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm start
```

Utilize Postman ou Insomnia para realizar
as requisições das rotas com a URL:

```endpoint
  http://localhost:3001/
```
